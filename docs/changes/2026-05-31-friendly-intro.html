Why

A new user landing on the GitHub repo or the docs site needed an obvious "start here" — a guided tour rather than a Sphinx API ref. This adds one hand-curated, self-contained HTML page that gives the toolbox a friendly entry point with runnable code, the nstat.extras tour, and the paper-example gallery.

What shipped

  • docs/intro.html — a self-contained page (embedded CSS, light/dark aware) with: hero, install commands, six capability cards each carrying a 5-10 line snippet using real public symbols, a tour of every nstat.extras subpackage, the paper-example thumbnail gallery (Examples 1-5), and a "Where to next" link panel. Published at /intro.html via html_extra_path.
