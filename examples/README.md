# Python nSTAT Examples

## Run the paper examples equivalent

```bash
cd python
python3 examples/nstat_paper_examples.py --repo-root ..
```

This is the Python equivalent of `helpfiles/nSTATPaperExamples.m` starting from Experiment 2.
Plot PNG files are generated by default under `plots/nstat_paper_examples/`. Use `--no-plots` to disable plotting.
